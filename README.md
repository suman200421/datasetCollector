## Introduction

The **Mobile Sensor Data Collection App** is a robust and extensible mobile application designed to collect, store, and export real-time data from device sensors. It enables researchers, engineers, and developers to gather high-quality sensor datasets for analysis, experimentation, machine learning, and IoT applications.

The app provides configurable sampling rates, session-based recording, and multiple export formats to ensure flexibility and scalability.
